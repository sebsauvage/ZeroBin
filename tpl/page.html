ZeroBin is a minimalist, open-source online pastebin where the server has zero knowledge of pasted data. Data is encrypted/decrypted in the browser using 256 bits AES. More information on the project page.

▶ Note: This is a test service: Data may be deleted anytime. Kittens will die if you abuse this service.
